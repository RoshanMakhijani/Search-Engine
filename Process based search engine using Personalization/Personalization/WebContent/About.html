Xplore is a search engine.
Process Based Search It introduces special search category called as process search.Each of the process based search is decomposed into basic steps and then individual search results for each steps is presented in a very intuitive manner. Volunteers can also add more processes, by simply clicking the appropriate link and help us improve our process database.
 
 
Personalization Xplore provides an option to personalize your search. Each interested user can create profiles for themselves. The results will be ranked according to their interests.
